Política de Privacidade — Stellaris Bot

Última atualização: 2026

1. Informações Coletadas

O Stellaris Bot coleta apenas dados necessários para funcionamento dentro do Discord. Os dados armazenados incluem:

  • ID do usuário do Discord
  • Contagem de mensagens enviadas
  • Tempo em canais de voz permitidos
  • Quantidade de estrelas e moedas do sistema interno
  • Registros de interações relacionadas à economia do servidor

Nenhum dado pessoal externo ao Discord é coletado.

2. Como os Dados São Utilizados

Os dados são utilizados exclusivamente para:

  • Funcionamento do sistema de recompensas
  • Rankings do servidor
  • Sistema de loja e cargos
  • Prevenção de abusos e manutenção do bot

3. Armazenamento de Dados

As informações são armazenadas localmente em banco de dados interno utilizado apenas pelo bot (arquivo database.json).

4. Compartilhamento de Dados

Nenhum dado é vendido, compartilhado ou distribuído a terceiros.

5. Segurança

Medidas razoáveis são aplicadas para proteger os dados armazenados, porém nenhum sistema é totalmente imune a falhas.

6. Retenção e Remoção de Dados

Administradores podem redefinir dados dentro do sistema. Usuários podem solicitar remoção de dados através do servidor oficial.

7. Alterações nesta Política

Esta política pode ser atualizada a qualquer momen
